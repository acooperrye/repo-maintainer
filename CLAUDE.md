A skill that manages GitHub repositories autonomously via Cowork. The architecture uses a limited-slip differential model for session state management, with 80+ discrete functional nodes across 16 clusters.

- Stubs are valid — don't fill them in by guessing.
- The mechanical metaphors are load-bearing. Work within them.
- One vertebra, one quadrant. Always.
- SPINE.md is the single source of truth. Reference files are wheels that receive torque from the spine.
